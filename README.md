## MacOS Preferences
- Set up finder sidebar
  - Drive
  - Desktop
  - Doenloads
  - Sites
- Keyboard
  - Key Repeat -> Fast
  - Delay Until Repeat -> Short
  - Disable "Correct spelling automatically"
  - Disable "Capitalize words automatically"
  - Disable "Add period with double-space"
  - Disable "Use smart quotes and dashes"
